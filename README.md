# SmartTask

**SmartTask** — fullstack demo task management application built with **React**, **TypeScript**, **Tailwind CSS**, and **.NET 10 backend** with **PostgreSQL**.

> ⚠️ Currently in development. A full-featured version with complete task management functionality will be available soon.

## Features

- Dynamic project cards with live data from PostgreSQL
- Responsive Hero section with hover effects
- Header with search, notifications, and Sign In / Sign Up buttons
- Fullstack architecture for real-time data fetching
- Easily extendable for task management functionality

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS, Heroicons
- **Backend:** .NET 10, C#, Entity Framework Core
- **Database:** PostgreSQL
